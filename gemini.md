I am presenting my project — a Web-Based College Predictor for Tamil Nadu Engineering Admissions, designed for both desktop and mobile users.

The main objective of this application is to help students predict suitable engineering colleges based on their Class 12 cutoff marks, similar to existing platforms but with enhanced insights and better usability.

This system replicates the functionality of existing cutoff-based prediction tools, but introduces several additional features to improve decision-making:

First, along with cutoff data, we include NIRF rankings from the previous year, helping students evaluate institutional quality.
Second, we calculate and display cutoff trends, specifically how much the cutoff has increased or decreased compared to previous years.
Third, we provide placement statistics, including average, median, and highest salary packages for each college.
Additionally, each college listing includes a direct redirect button to the official university website.

The application also includes a personalized user interaction system, where students can register for upcoming one-on-one guidance sessions through a floating popup. The collected data—such as name, school, email, and mobile number—is securely stored in Google Drive for further communication.

From a technical standpoint:

The frontend is built using Antigravity tools for rapid UI development.
The application is hosted on Vercel for high performance and scalability.
The backend integrates web scraping to collect real-time data from Tamil Nadu engineering colleges across all districts.

The system is optimized for fast response times, ensuring smooth user experience across devices.

In conclusion, this project aims to simplify the college selection process by combining accurate data, predictive insights, and user-friendly design in a single platform.
